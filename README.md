

# Исследование компромисса между точностью и приватностью в real-time прогнозировании трафика с использованием синтетических потоков траекторных данных

## Описание проекта

В работе исследуется возможность использования синтетических траекторных данных в системах потоковой аналитики транспортной мобильности умного города. На основе датасета T-Drive Taxi Trajectories (Microsoft Research, ~10 357 такси, Пекин) проведена сравнительная оценка трёх архитектурных сценариев: обучение на реальных данных, исключительно на синтетических (CTGAN) и на гибридных. Оцениваются точность прогнозирования средней скорости трафика (MAE, RMSE, R²), задержка потоковой обработки и метрики приватности (k-анонимность, риск повторной идентификации).

## Датасет

Используется датасет T-Drive Taxi Trajectories, опубликованный Microsoft Research: https://www.kaggle.com/datasets/arashnic/microsoft-t-drive-taxi-trajectories

Датасет содержит GPS-траектории 10 357 такси города Пекина за период с 02 по 08 февраля 2008 года (~15 млн точек наблюдений). Каждая запись включает: taxi_id, datetime, longitude, latitude. Для исследования использована выборка 500 транспортных средств (~2–3 млн точек после фильтрации).

## Архитектурные сценарии

Сценарий A — только реальные данные (baseline): модель обучается и тестируется на реальных агрегатах GPS-траекторий.

Сценарий B — только синтетические данные (CTGAN): модель обучается на данных, сгенерированных CTGAN, и тестируется на реальных данных (протокол TSTR).

Сценарий C — гибрид 50/50: обучающее множество формируется из равных долей реальных и синтетических данных, тестирование на реальных данных.

## Результаты

Точность прогнозирования (целевая переменная — средняя скорость трафика, км/ч):

| Сценарий | MAE | RMSE | R² | R² / Baseline |
|---|---|---|---|---|
| A: Только реальные (baseline) | 3.87 | 5.21 | 0.847 | 100% |
| B: Только синтетика (CTGAN) | 4.53 | 6.08 | 0.779 | 92.0% |
| C: Гибрид 50/50 | 4.02 | 5.44 | 0.831 | 98.1% |

Задержка обработки: реальные данные — 1.83 мс, синтетические — 4.67 мс (с учётом накладных расходов CTGAN-инференса). Оверхед модели без учёта генерации — менее 10%.

Приватность: k_mean для реальных данных — 3.24, для синтетических — 4.87. Доля уникальных комбинаций квазиидентификаторов снижена на 52% (0.412 → 0.198). Риск повторной идентификации по методу ближайшего соседа — 0.05.
